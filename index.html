Kaggle: Competições de Machine Learning

By António Cruz on January 5, 2017

Muita da informação presente neste post foi diretamente traduzida da wikipedia. Para quê inventar a roda? O que é a Kaggle Em 2010, a Kaggle foi fundada como uma plataforma para competições de modelos preditivos e analítica, nas quais empresas e pesquisadores disponibilizam os seus dados e analistas de dados de todo o mundo competem para produzir os melhores modelos.
